Design Philosophy

设计哲学

以端侧为底座、以隐私为前提、以极客工具为触感、以跨平台可用性为目标，构建稳定且可扩展的语音上下文体验。

端侧

语音识别、翻译与合成尽量在本地完成，低延迟、可离线、可持续运行。

隐私

敏感音频与文本保留在本机，云端能力仅在授权后调用，数据可追溯。

极客瑞士军刀

可插拔模型、可编排流程、可拓展插件，让重度用户也能高效掌控。

易用，跨多平台

桌面主战场，兼顾 macOS 与 Windows，减少学习成本，随时开机即用。

Quick Start

快速开始

两步完成端侧配置，既可下载模型离线使用，也可对接 API 立即接入云端能力。

两条路径，同一体验

离线模型保证响应速度，云端接口补充高阶能力。可按场景自由切换。

本地模型 API 接入 一键启动
Step 01 · 模型 / API

下载模型或配置 API

  • 下载 SenseVoice / TTS 本地模型并指向目录。
  • 或填写 OpenAI 兼容 API Base URL 与密钥。
  • 确认实时字幕、翻译与合成的链路状态。
Step 02 · 安装启动

安装并启动无界音流

  • 安装桌面客户端，首次启动完成基础设置。
  • 按下快捷键开始录音，实时输出字幕。
  • 根据场景开启翻译、纠错与语音合成。
