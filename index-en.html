Design Philosophy

Design Philosophy

Built on on-device foundations, privacy by default, geek-tool flexibility, and cross-platform usability for a stable, extensible voice context experience.

On-device

Speech recognition, translation, and synthesis run locally with low latency, offline ability, and continuous availability.

Privacy

Sensitive audio and text stay on your device; cloud capability is only used with explicit permission.

Geek Swiss Army Knife

Pluggable models, composable flows, and extensible plugins for power users and complex workflows.

Easy, cross-platform

Desktop-first across macOS and Windows with minimal learning curve and instant readiness.

Quick Start

Quick Start

Two steps to configure on-device usage, with optional API access for cloud capabilities.

Two paths, one experience

Offline models ensure speed; cloud APIs add advanced capabilities. Switch freely by scenario.

Local models API access One-click start
Step 01 · Model / API

Download models or configure API

  • Download SenseVoice / TTS local models and point to the folder.
  • Or provide an OpenAI-compatible API Base URL and key.
  • Validate realtime captions, translation, and synthesis pipelines.
Step 02 · Install & Launch

Install and run Boundless Flow

  • Install the desktop client and complete initial settings.
  • Press the shortcut to start recording and stream captions.
  • Enable translation, proofreading, and synthesis as needed.
